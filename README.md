# Lunariella

## ¿Quién diablos soy?

풀스택이라고 쓰고 '이것저것 다 한다'고 읽는다.
프론트도 하고 백도 하고 DB도 만지고 인프라도 본다.
잘하는 건 아니고, 그냥 다 해본 것뿐이다.

코드를 오래 쓰다 보니 이제는 뭐가 좋은 코드인지 모르겠다.
돌아가면 좋은 코드고, 안 터지면 훌륭한 코드다.

## Mi arsenal

**Java** - 제일 오래 썼다. 장황하지만 믿을 수 있다.  
**Python** - 빠르게 뭔가 만들 때. 들여쓰기는 여전히 신경 쓰인다.  
**Node.js** - 프론트 개발자들이 좋아하니까. callback hell은 이제 익숙하다.

가끔 다른 것도 만진다. 필요하면 뭐든 배운다.

## Este repositorio

새벽에 떠오른 아이디어의 묘지.
"이거 만들면 대박일 텐데" 싶어서 시작했다가 다음 날 "이게 뭐지?" 하는 것들.

여기 있는 코드들:
- 아무도 안 쓸 자동화 도구
- 끝까지 완성 못한 사이드 프로젝트
- "일단 돌아가게만" 만든 프로토타입
- 술 마시고 짠 코드 (sorprendentemente funciona)

가끔은 쓸만한 게 나온다. 정말 가끔.

## Filosofía personal
```javascript
function desarrollar(idea) {
  try {
    return '완벽한 코드';
  } catch (realidad) {
    return '돌아가는 코드';
  }
}
```

완벽을 추구하다가 아무것도 못 만드는 것보다,
대충 만들어서 돌아가는 게 낫다는 걸 배웠다.

*"No es un bug, es una característica no documentada."*  
(버그가 아니다, 문서화 안 된 기능이다.)
